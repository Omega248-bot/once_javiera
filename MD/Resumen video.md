# Resumen
## 1.¿Como funciona el internet?

Para comprender el funcionamiento del internet de manera más sencilla no lo asociemos de manera compleja, sino como un sistema de comunicación basado en reglas.

### Protocolos TCP/IP

### - IP 
Se encarga de asignar direcciones a los dispositivos y de enviar los datos en paquetes con dirección de origen y destino, como si fueran sobres. 
### - TCP 
Garantiza que esos datos lleguen completos y en orden, dividiéndolos en partes y solicitando el reenvío si alguna se pierde.
### - DNS:
Funciona como una agenda que traduce los nombres de páginas web **(como google.com)** en direcciones **IP** numéricas que las computadoras usan para comunicarse.
### - DHCP:
Asigna automáticamente una dirección IP y otros datos necesarios **(como el router y el DNS)** cuando un dispositivo se conecta a una red, facilitando la conexión sin configuración manual.

## 2. Protocolo HTTP y la Web

El protocolo **HTTP** es el idioma que usan los navegadores y los servidores para comunicarse cuando navegas por internet. Cada vez que entras a una página, tu navegador envía una solicitud y el servidor responde con la información necesaria para mostrar el sitio.La dirección que escribes, llamada **URL**, tiene tres partes principales: el esquema **(HTTP o HTTPS)**, el dominio **(nombre del sitio)** y la ruta **(la sección o archivo específico que quieres ver)**. **HTTP** también utiliza acciones como **GET**, para pedir información, y **POST**, para enviar datos, como en un formulario. Cuando el servidor responde, envía un código que indica qué ocurrió: 200 significa que todo salió bien, 404 que la página no existe y 500 que hubo un error interno.
